CSS 선택자

  • 전체 선택자

    • 모든 요소를 선택한다.
    • 우선 순위가 0점이다.
    • 문서 전체에 공통적으로 적용할 때 사용한다.

  • 타입(태그)

    • 적용하고자 하는 태그의 종류로 선택한다.
    • 우선 순위가 1점이다.
    • 문서 내에 있는 해당 태그의 모든 요소에 적용된다.

  • 클래스 선택자

    • .클래스명 : 요소의 클래스 속성의 값으로 선택한다.
    • 우선 순위가 10점이다.
    • 문서 내에 있는 해당 태그의 모든 요소에 적용된다.
    • 공통적인 스타일을 지정하고 적용하고자 하는 태그에 class를 지정하면 된다.
    • 띄어쓰기를 통해 여러개의 클래스를 가지는 것도 가능하다.
      => 여러개의 클래스를 정의하고 조합하여 사용할 수 있다.

  • 아이디 선택자

    • # 아이디명 : 요소의 id 속성의 값으로 선택한다.
    • 우선 순위가 100점이다.
    • 문서 내에 있는 id 값은 중복되면 안된다. 즉, 유일해야 한다. * CSS 적용이 안 되는 건 아님, 다만 사회적 약속
    • 공통적인 스타일을 지정하고 적용하고자 하는 태그에 class를 지정하면 된다.
    • id값은 여러개 부여하는 것이 불가능하다.

  • 일치 선택자

    • 여러 조건을 모두 충족하는 요소에만 적용한다.

  • 자식 선택자

    • 자식(바로 밑의 후손) 요소에만 적용한다.

  • 요소 후손 선택자

    • 후손 요소에 모두 적용한다.

  • 인접 형제 선택자, 일반 형제 선택자

    • + : 인접한 형제에 적용한다.
    • 인접 형제 선택자는 바로 옆에 있는 형제 요소(바로 다음에 오는) 하나에만 적용한다.
    • ~ : 이후의 모든 형제 요소에 적용한다.

  • 속성 선택자

    • 요소[속성명] :[]안의 속성을 가진 요소에만 적용한다.
    • 요소[속성명=값] : 특정한 속성값을 가진 요소에만 적용한다.

  • 가상(Pseudo: 의사) 클래스선택자

    • 마우스 호버 등 특정 상황에서만 css 적용 할 때 사용한다.
    • 실제로 클래스를 지정하진 않지만 가상으로 지정한 것처럼 동작한다.

  • 가상(Pseudo:의사) 요소 선택자

    • 문서에 실제로 작성되지 않았지만 디자인적으로 요소를 추가할 때 사용한다.
    • 문서 내에 불필요한 내용이지만 디자인적으로 요소가 필요할 때 사용한다.
